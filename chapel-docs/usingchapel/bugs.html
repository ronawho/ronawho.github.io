Reporting Chapel Issues¶

Public Issues¶

Chapel uses GitHub Issues to track public issues. If you have a bug report, feature request, or other issue to report, please use the “New Issue” button on that page to file it.

Public issues that predate our issue tracker can be found in the (retired) STATUS.devel file and the (retired) chapel-bugs mailing list archives.

Private Issues¶

Chapel accepts private issue reports by email.
