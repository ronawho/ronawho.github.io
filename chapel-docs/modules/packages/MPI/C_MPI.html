C_MPI¶

Usage

use MPI.C_MPI;

or

import MPI.C_MPI;

Low level MPI bindings in Chapel.

The MPI module wraps this submodule to provide a higher level API. For documentation of the routines in this modules, please refer to your MPI documentation.
